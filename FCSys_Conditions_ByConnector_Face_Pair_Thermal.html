FCSys.Conditions.ByConnector.Face.Pair.Thermal.PartialCondition

Partial function to select a thermal quantity

Information

This function takes as inputs all the efforts and flows of the associated connector. It should be extended to add an algorithm that maps these inputs to a single value.

Extends from Modelica.Icons.Function (Icon for functions).

Inputs

TypeNameDefaultDescription
DensityDeltarho Difference in density (Δρ) [N/l3]
CurrentNdot Diffusion current (Ṅ) [N/T]
VelocityDeltaphi[Orient] Difference in velocity (Δφ) [l/T]
ForcemPhidot[Orient] Non-equilibrium force (mΦdot) [l.m/T2]
TemperatureAbsoluteDeltaT Difference in temperature (ΔT) [l2.m/(N.T2)]
PowerQdot Rate of thermal conduction (Q̇) [l2.m/T3]

Outputs

TypeNameDescription
RealxValue of condition

Modelica definition

partial function PartialCondition 
  "Partial function to select a thermal quantity"
  extends Modelica.Icons.Function;

  // Material
  input Q.Density Deltarho 
    "Difference in density (Δρ)";
  input Q.Current Ndot "Diffusion current (Ṅ)";

  // Translational
  input Q.Velocity Deltaphi[Orient] 
    "Difference in velocity (Δφ)";
  input Q.Force mPhidot[Orient] 
    "Non-equilibrium force (mΦdot)";

  // Thermal
  input Q.TemperatureAbsolute DeltaT 
    "Difference in temperature (ΔT)";
  input Q.Power Qdot "Rate of thermal conduction (Q̇)";

  output Real x "Value of condition";
end PartialCondition;
