FCSys.WorkInProgress.Blocks.Routing.Duplicate

Duplicate a vector signal n_dup times

Information




This blocks computes output y (element-wise) as
integral of the input u multiplied with
the gain k:



         k
     y = - u
         s





It might be difficult to initialize the integrator in steady state.
This is discussed in the description of package
Continuous.

block Duplicate "Duplicate a vector signal n_dup times"
  extends FCSys.BaseClasses.Icons.Blocks.Continuous;
  extends FCSys.WorkInProgress.Blocks.BaseClasses.MI;
  extends FCSys.WorkInProgress.Blocks.BaseClasses.MO(final n_out=n_in*n_dup);
  parameter Integer n_dup=1 "Number of duplications";

equation 
  for i in 1:n_dup loop
    y[n_in*(i - 1) + 1:n_in*i] = u;
  end for;
end Duplicate;
